# Valentine Website

This is a single-page romantic website built with React (via CDN) and plain HTML/CSS/JS, ready to deploy on Netlify.

## What's Included
- Soft pink + pastel purple theme
- Floating hearts background
- Cute illustrations that change after every "No"
- Button resizing behavior (Yes grows, No shrinks)
- Full-screen success message with soft romantic music
- State persistence using `localStorage`

## Run Locally
Open `index.html` in a browser.

## Deploy to Netlify
1. Drag and drop the project folder into Netlify.
2. Or connect the Git repo and deploy.

Netlify uses `netlify.toml` to publish the root folder.
